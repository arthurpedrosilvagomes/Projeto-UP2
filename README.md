# Aplicativo de lanche de cantina

## O aplicativo contém:

 * Interface do usuário em console;
 * Funcionalidades bem definidas;
 * Função cardápio desenvolvido;
 * Função de inserção de saldo desenvolvido;
 * Vetor pedido para customização de pedido desenvolvido;
 * Função de confirmar ou eliminar pedido desenvolvido;

## Correções:

 * Ao invés de varias Arryas para cada tipo de lanche, agora só tem um para todos; 
 * O usuário pode customizar seu pedido e pagar o montante de cada lanche; 
 * Agora, o carrinho e o cardápio são vetores para que possam ser alterados; 

## Adições:

 * Sessão para servidor (funcionário);
 * Funcionalidade de alterar o cardápio (desenvolvido);
 * Antes de efetuar a compra do pedido, o cliente informa seu CPF;
 * Agora é possível analisar vendas;
